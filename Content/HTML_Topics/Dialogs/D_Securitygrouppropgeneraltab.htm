Dialoogvenster Security groep Eigenschappen - tabblad Algemeen

U gebruikt het Algemeen tabblad van het Security groep Eigenschappen dialoogvenster om de naam, en de toegangsrechten die leden van de security groep hebben over bepaalde aspecten van het project, te bewerken.

U kunt u de leden van de security groep speciale rechten geven voor balken en taken met een bepaalde code, uit een gespecificeerde "security bibliotheek", als aanvulling op reeds gedefinieerde toegangsrechten die u al via Kaart Toegangsrechten tabblad gedefinieerd heeft.

Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Naam Voer de naam van de security groep in.
Rechten bibliotheek

Om toegangsrechten voor een bepaalde code of codebibliotheek toe te wijzen (als aanvulling op kaart toegangsrechten), selecteert u de betreffende codebibliotheek in dit veld. U kunt dan de toegangsrechten voor een bepaalde code of voor de hele bibliotheek specificeren op het Kaart Toegangsrechten tabblad.

 

Bijvoorbeeld, u heeft een codebibliotheek genaamd Mensen die een code bevat voor iedere medewerker in het bedrijf, zodat u elke gebruiker volledige toegang zou kunnen geven tot balken, taken of allocaties die hun code hebben. Deze rechten komen bovenop de toegangsrechten die per kaart zijn ingesteld. Een gebruiker kan dus alleen-lezen rechten hebben voor een kaart, maar volledige toegang tot balken, taken of allocaties met zijn code binnen de kaart.

 

 

Wanneer u een codebibliotheek selecteert in dit veld, moet u het Bibliotheek Toegangsrechten tabblad gebruiken om de bibliotheek alleen-lezen te maken, om de leden van de security groep te verhinderen om de toegangsrechten te wijzigen voor de bibliotheek.

 

Specificeren van unieke toegangsrechten over een "security bibliotheek"

Leden van de security groep kunnen alleen gegevens archiveren van kaarten en samenvattingsgroepen waarvoor zij Lezen rechten hebben. Dit is gedefinieerd op het Kaart Toegangsrechten tabblad van het dialoogvenster. U kunt alleen gegevens archiveren en terugzetten wanneer u gebruik maakt van Enterprise.
