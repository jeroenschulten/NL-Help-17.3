Options dialog - File Locations tab

You use the File Locations tab of the Options dialog to specify the location of the various types of file that relate to . The tab lists the file types and the location in which each type of file is stored. Depending on the type of file whose location details you change, you may have to restart for the changes that you make to take effect.

As well as specifying fixed file locations - for example 'C:\Data\Documents\', you can specify file locations using the '%userprofile%' environment variable. This means that you can specify file locations that are within a user's profile folder, without having to specify the user's name. For example, you can specify a file location of '%userprofile%\Templates\', or '%userprofile%\Documents', and these will be resolved into the individual user's Templates or Documents folder.

You can also use the '%userprofile%' environment variable to specify file locations that exist in a user's OneDrive, or in SharePoint. For example, if you specify a file location of '%userprofile%\OneDrive\Documents\Powerproject\Projects\', this will be resolved to the folder in the user's OneDrive (note that the name of the user's OneDrive may differ from 'OneDrive').

If you specify a location for a file type that is on a network drive rather than on your local computer, we recommend that for best performance, you map the network drive on your computer. For example, if you set a location for your live projects that is on a network drive, mapping that drive on your computer makes it easier to open projects from and save projects to that folder.

Explanation of the fields and buttons on the dialog
Field/Button Description
File locations A list of the file types used by and the location in which each type of file is stored. Click a file type to select it in the list.
Open Folder Click this button to view the selected folder in Windows Explorer.
Default Click this button to restore the location of the selected file type back to the default setting.
Modify Click this button to change the path of the selected file type.
Clear

Click this button to clear the path of the selected file type (macro files only).

 
Only use the given file locations for borders and templates

In the case of project templates and border files, you can specify whether should search not only in the folders that are specified on this dialog, but also in other locations - for example, in the folder in which is installed. You may want to prevent templates and borders in other locations from being presented to you each time you create a new project or print, to ensure that you use only the appropriate templates and borders.

 

Select this check box to prevent additional templates and borders from being presented to you, or clear the check box to specify that templates and borders in other locations should be presented to you alongside the templates and borders that are stored in the folders specified on this dialog.

... change the location of a file type?
  1. Select the type of file whose location you want to change.
  2. Click Modify. You see the Default Path for Files dialog.
  3. Browse for the location in which you want to save files of the selected type and click OK to return to the Options dialog.
 ... restore the location of a file type back to the default setting?
  1. Select the type of file whose location you want to restore back to the default setting.
  2. Click Default.

You cannot restore the location of macro files back to a default setting, as macro files do not have a default location.

 ... clear the location of a file type?
  1. Select the type of file whose location you want to clear (you can only clear the location of macro files).
  2. Click Clear.

 ... specify whether project templates and borders in shared locations should also be used?

Select the Only use the given file locations for borders and templates check box to prevent additional templates and borders in other locations (for example, in the folder in which is installed) from being presented to you, or clear the check box to specify that templates and borders in other locations should be presented to you alongside the templates and borders that are stored in the folders specified on this dialog.
