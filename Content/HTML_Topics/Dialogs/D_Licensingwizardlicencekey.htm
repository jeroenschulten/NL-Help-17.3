Wizard Licentie - Licentie Code pagina

U gebruikt de Licentie Wizard om uw Elecosoft software te licentiëren. De Licentie Wizard bestaat uit een aantal stappen. Elke stap wordt duidelijk uitgelegd in de schermen van de wizard.

U gebruikt deze pagina van de Licentie Wizard om de licentiecode die u van Elecosoft BV hebt ontvangen in de wizard te plakken.

Wanneer u de licentiecode van Elecosoft BV hebt ontvangen, kopieert u deze naar het Windows® Klembord klikt u op Plakken om de code in deze pagina van de wizard te plakken. Klik vervolgens op Voltooien. De software is nu gelicentieerd op deze computer.
