Wizard Licentie - Genereer een Licentie Aanvraag Code pagina

U gebruikt de Licentie Wizard om uw Elecosoft software te licentiëren. De Licentie Wizard bestaat uit een aantal stappen. Elke stap wordt duidelijk uitgelegd in de schermen van de wizard.

Deze pagina van de Licentie Wizard toont informatie die gevraagd wordt door Elecosoft BV om u van een licentiecode te kunnen voorzien - een 'licentie aanvraag code'. U moet de licentie aanvraag code van deze computer naar Elecosoft Support sturen. U kunt dit doen in een e-mail bericht, of wanneer uw computer geen e-mail heeft, via een andere methode, bijvoorbeeld een floppy disk.

Klik op e-mail om de licentie aanvraag code over te brengen naar een e-mail bericht, of klik op Kopiëren om deze te kopiëren naar het Windows® Klembord. U kunt dan de e-mail naar de andere computer sturen. Of breng de licentie aanvraag code over via een TXT bestand op een floppy disk.

Wanneer u de licentieaanvraagcode verzonden hebt naar Elecosoft, zal de Support Afdeling u een licentiecode sturen, waarmee u de software kunt licentiëren.

Stuur de licentie aanvraag code naar Elecosoft Support, klik vervolgens op Volgende om naar de volgende pagina van de licentie wizard te gaan.
