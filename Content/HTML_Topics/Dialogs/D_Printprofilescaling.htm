Print Profile Properties dialog - Scaling tab

You use the Scaling tab of the Print dialog to specify a horizontal and vertical scale for the printed output, should you want to override the default horizontal and vertical scale settings when a particular print profile is used. Specifying a scale may be useful if you want to produce a printout with precise layout settings. However, in most circumstances you will obtain the optimum printout with a sensible aspect ratio by using the settings defined in the Format Bar Chart dialog. You also use this tab to specify the way in which you want a border file to surround the output, and to specify the way in which the output is paginated when a particular print profile is used.
