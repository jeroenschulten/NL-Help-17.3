Timeline Date Range dialog

You use the Timeline Date Range dialog to specify the date range that should be covered by the project timeline.

How to access the Timeline Date Range dialog

To access the Timeline Date Range dialog:

  1. Click in the timeline pane.
  2. On the Format tab, in the Show/Hide group, click Timeline Date Range.
 Explanation of the fields on the dialog
Field Description
Use project start and finish dates Click this radio button to specify that the project timeline should cover the entire date range of the project, from the project start date to the project finish date.
Custom dates Click this radio button to specify that the project timeline should cover a date range other than from project start to finish. You use the following fields to specify the date range.
Start If you want the project timeline to cover a custom date range, enter the start date.
Finish If you want the project timeline to cover a custom date range, enter the finish date.
... make the project timeline cover the entire date range of the project?

Click the Use project start and finish dates radio button.

 ... make the project timeline cover a custom date range?
  1. Click the Custom dates radio button.
  2. Enter the date at which the custom date range should start in the Start date field.
  3. Enter the date at which the custom date range should end in the Finish date field.
