Task Activity Template Properties dialog

You use the Task Activity Template Properties dialog to create templates of commonly-used task activities, which can then be applied to tasks. You can set up any number of task activity templates, each of which contains a predefined list of task activities.

How to access the Task Activity Template Properties dialog

To access the Task Activity Template Properties dialog, either:

  • Right-click a task activity template in Library Explorer and select Properties.
  • Right-click the Activities grid, in the 'Task Activities' group of fields on the Task tab of the Bar and Task Properties dialog, and select Create Activity Template.
 Explanation of the fields and buttons on the dialog
Field/Button Description
Name Enter a descriptive name for the task activity template.
Activities

This grid displays a list of all task activities in the template. For each task activity, specify the following information in the columns of the grid:

  • Activity - Enter a name to identify the task activity. Each task activity name in the grid must be unique.
  • Weight - Specify a factor by which the progress of the task activity should be multiplied to affect its impact on progress of the task. The greater the weight of a task activity, the greater proportion of the task duration it will contribute to as it is completed. If a task activity has a weight of zero, it will not contribute towards the progress of the task at all.
Add Click this button to add a new task activity to the grid. The new activity appears at the bottom of the list of task activities.
Remove Click this button to delete the selected task activity from the grid.
Move Up Click this button to move the selected task activity up one position in the grid.
Move Down Click this button to move the selected task activity down one position in the grid.
