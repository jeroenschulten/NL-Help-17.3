Backstage view - Open tab

You use the Open tab of the Backstage view to open projects and - if you are using - to specify the location of the Enterprise server.

If more than one user has been set up in the project that you are opening and you know the user name and password of multiple users, you can open a project using a different user's user name and password.

Explanation of the fields and buttons on the Open tab
Field/Button Description
Recent

Click this button to open a project that you have accessed recently. A list of recently-accessed projects appears. Click a project to open it.

 
This PC Click this button to open a project that is located in a folder on your PC or on a mapped network drive. A navigable folder list appears. Navigate around the folder list and click a project to open it.
Enterprise

Click this button to open a project that is stored in a server-mounted data source. A navigable list of data sources and projects appears. Navigate around the data sources and projects and click a project to open it.

 

You also click Enterprise to access the Server Locator button, which you use to specify the location of your servers.
