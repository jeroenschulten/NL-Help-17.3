Microsoft Project Export Wizard - Project Milestones page

You use this page of the Microsoft Project Export Wizard to specify how to handle any finish milestones in the project. Microsoft Project does not support finish milestones.

Explanation of the fields on the dialog
Field Description
Change all Finish milestones into Start milestones

Click this radio button to change all finish milestones into start milestones, both in the exported data and in the original project.

 
Change any Finish milestones linked SF/FF into Start milestones

Click this radio button to change all finish milestones that have one or more Start-to-Finish or Finish-to-Finish incoming or outgoing links into start milestones in the exported data, both in the exported data and in the original project.

 
Do not adjust project data

Click this radio button to change all finish milestones into start milestones in the exported data, but leave all finish milestones in the original project as they are.

 
... specify how to handle any finish milestones in the project?
  1. Click the appropriate radio button to specify what you want to do with any finish milestones:
    • Click Change all Finish milestones into Start milestones to change all finish milestones in the project into start milestones, both in the exported data and in the original project.

    • Click Change any Finish milestones linked SF/FF into Start milestones to change all finish milestones that have one or more Start-to-Finish or Finish-to-Finish incoming or outgoing links into start milestones in the exported data, both in the exported data and in the original project.

    • Click Do not adjust project data to change all finish milestones into start milestones in the exported data, but leave all finish milestones in the original project as they are.
