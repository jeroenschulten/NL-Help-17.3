Microsoft Project Export Wizard - Data Export Limitation Notifications page

This page of the Microsoft Project Export Wizard warns you of possible data loss or differences in project structure that may be apparent following an export. This can occur if the project you are exporting contains certain elements that are not supported by Microsoft Project.

Explanation of the warnings on the dialog

The warnings on this page of the wizard are categorised as follows:

Icon Description
These warnings are for information only. They inform you of possible differences in project structure following the export.
These warnings are more serious. They inform you of possible data loss following the export.
