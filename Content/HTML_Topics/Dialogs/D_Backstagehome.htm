Backstage view - Home tab

U gebruikt het tabblad Start van de Backstage view om toegang te krijgen tot recente projecten en tot 'vastgepinnde' projecten. Ook kunt u er nieuwe projecten creëren, projecten afdrukken en toegang krijgen tot help en support.

Explanation of the fields and buttons on the Home tab
Veld/Knop Omschrijving
New Project

Klik op deze knop om toegang te krijgen tot het tabblad Nieuw van de Backstage view, dat u kunt gebruiken voor het creëren van nieuwe projecten.

 

Backstage view - Nieuw tabblad

Import Project

Klik op deze knop om toegang te krijgen tot het tabblad Import/Export van de Backstage view, die u kunt gebruiken om projecten te importeren uit verschillende bestandsindelingen en om projecten te exporteren naar verschillende bestandsindelingen, zodat uw projecten kunnen worden geopend in andere projectmanagement toepassingen.

 

Backstage view - Import/Export tabblad

Print

Klik op deze knop om toegang te krijgen tot het tabblad Afdrukken van de Backstage view, die u kunt gebruiken om een voorbeeld te bekijken en de huidige weergave van het project dat momenteel actief is af te drukken.

 

Backstage view - Afdrukken tabblad

Help Gids/Support

Klik op deze knop om toegang te krijgen tot de Help-tab van de Backstage view, die u kunt gebruiken om toegang te krijgen tot Help over het gebruik van , om toegang te krijgen tot de tutorial en om de website van Elecosoft te openen. Als u of uw organisatie uw eigen documentatie beschikbaar heeft gemaakt in , kunt u via dit tabblad ook toegang krijgen tot dergelijke aanvullende documentatie.

 

Backstage view - Help tabblad

Zoek Voer een alfanumerieke tekenreeks in dit veld in om ervoor te zorgen dat de onderstaande lijst met projecten alleen projecten weergeeft waarvan de bestandsnaam de door u ingevoerde tekst bevat. U kunt de lijst bijvoorbeeld filteren om alleen projecten weer te geven die het woord 'Ontwikkeling' in hun bestandsnaam bevatten. De lijst wordt direct gefilterd wanneer u tekst invoert in dit veld.
Recent Klik op deze knop om display a list of up to 16 of the most recently-accessed projects. You can filter the projects that appear in the list by entering text into the Search field. Click a project to open it.
Pinned

Klik op deze knop om display a list of any projects that you have 'pinned' (in the list of recent projects that appears on the Open tab) so that they always appear in this list. You can filter the projects that appear in the list by entering text into the Search field. Click a project to open it.

 

Backstage view - Open tab

... create a new project?

Click New Project to access the New tab of the Backstage view, which you can use to create new projects.

 ... import a project from a different file format?

Click Import Project to access the Import/Export tab of the Backstage view, which you can use to import projects from a variety of different file formats and to export projects into a variety of different file formats, so that your projects can be opened in other project management applications.

 ... print the current project?

Click Print to access the Print tab of the Backstage view, which you can use to preview and print the current view of the project that is currently active.

 ... access Help and support?

Click Help Guide/Support to access the Help tab of the Backstage view, which you can use to access Help on using , to access the tutorial and to launch Elecosoft's website.

 ... open a recently-accessed project?

Click Recent to display a list of up to 16 of the most recently-accessed projects. You can filter the projects that appear in the list by entering text into the Search field. Click a project to open it.

 ... open a 'pinned' project?

Click Pinned to display a list of any projects that you have 'pinned' (in the list of recent projects that appears on the Open tab) so that they always appear in this list. You can filter the projects that appear in the list by entering text into the Search field. Click a project to open it.
