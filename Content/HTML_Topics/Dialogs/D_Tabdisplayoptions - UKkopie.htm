Tab Display Options dialog

You use the Tab Display Options dialog to configure the way in which 's tabs are displayed. displays each view in a different tab, and you can switch between views by clicking the various tabs.

How to access the Tab Display Options dialog

To access the Tab Display Options dialog, click the Tab Options button on the General tab of the Options dialog.

 Explanation of the fields on the dialog
Field Description
Location

Specify whether tabs should be displayed above or below the views to which they relate.

 

Configuring how views are displayed in tabs

Tab mode

Specify whether to display an additional tab that displays the project window.

 

The project window displays a project tree and enables you to browse the project's hierarchy when the project view is not displayed in other views. You may, or may not wish to display it.

 

Note that if you change the setting in this field, your changes do not take effect until you have closed and restarted .
Use tab colouring Select this check box to apply colouring to the tabs that appear at the top of each view.

 
... specify where the tabs appear in relation to the views?

Specify where the tabs should appear in relation to the views in the Location field. Select:

  • Top to display the tabs above the views.
  • Bottom to display the tabs below the views.
 ... specify whether to display an additional tab that displays the project window?

Specify the way in which views are displayed in tabs in the Tab mode field. Select:

  • Tabs and no project tab to display each view in a separate tab, without including a tab that displays the project window.
  • Tabs and project tab to display each view in a separate tab, including a tab that displays the project window. If you select this option, the project window tab is always displayed.

Select the Use tab colouring check box to apply colouring to the tabs that appear at the top of each view.
