Library Explorer

Projects can contain a large number of objects such as resources, calendars, codes, cost centres, currency units and users. stores these objects in libraries.

The Library Explorer lists all the objects within the current project. The Library Explorer has two panes:

  • The left pane lists the libraries of objects that the project can contain.
  • The right pane lists the objects within the selected library. You can create, edit and delete objects in this pane.
