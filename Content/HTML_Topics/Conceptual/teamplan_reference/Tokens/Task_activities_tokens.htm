Taak activiteiten velden

Voor elk van de onderstaande velden, wordt de volgende informatie gegeven:

  • Omschrijving - de informatie die in het veld wordt weergegeven wanneer dit gekoppeld is aan de tabel of aan een tekstnotitie.
  • Parameters - elke extra optionele parameters die u kunt toevoegen om de informatie die het veld weergeeft of de manier waarop die informatie getoond wordt te wijzigen.
  • Aanpasbaar? - of het veld aanpasbaar is wanneer het wordt weergegeven in een tabelkolom. Als een veld in de tabel aanpasbaar is, kunt u de gegevens die in de kolom worden getoond wijzigen, om de eigenschappen van de overeenkomstige balk, taak of toewijzing aan te passen.

VoltooideTaak Activiteiten

Omschrijving

The number of task activities on a task that are 100% complete.

 

For more information, refer to Breaking down tasks into smaller steps using task activities.
   

Parameters
  • Data - of u de gegevens wilt weergeven over de huidige data of over een specifieke baseline.
  • Datums - specificeer indien gewenst een datumdomein.
  • Specifieke taak - specificeer indien gewenst de specifieke taak of taken op een multi-taak balk waarop u wilt dat het veld betrekking heeft.

Taak Activiteiten

Omschrijving

The number of task activities on a task, regardless of their state of progress, followed by the word 'Activities'.

 

This field displays the same value for each task as the 'Task Activity Count' field, but with the word 'Activities' appended to the value. You may like to use this field to indicate the number of task activities in the bar chart, rather than using the 'Task Activity Count' field, as the this field gives context to the number that is displayed in the bar chart.

 

For more information, refer to Breaking down tasks into smaller steps using task activities and Displaying bar and task information in the bar chart.

Parameters
  • Data - of u de gegevens wilt weergeven over de huidige data of over een specifieke baseline.
  • Datums - specificeer indien gewenst een datumdomein.
  • Specifieke taak - specificeer indien gewenst de specifieke taak of taken op een multi-taak balk waarop u wilt dat het veld betrekking heeft.

Aanpasbaar?
