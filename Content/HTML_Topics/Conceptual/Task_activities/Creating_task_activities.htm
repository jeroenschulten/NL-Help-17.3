Creating task activities

To create activities for a task, breaking it down into smaller steps:

  1. Right-click the task that you want to break down into smaller steps using task activities and select Properties. The Bar and Task Properties dialog appears.
  2. Click Add to add a new task activity to the grid. The new task activity appears at the bottom of the list of task activities.
  3. Enter a descriptive name to identify the task activity in the Name column. Each task activity name in the grid must be unique.
  4. Specify a factor by which the progress of the task activity should be multiplied to affect its impact on progress of the task in the Weight column. The greater the weight of a task activity, the greater proportion of the task duration it will contribute to as it is completed. If a task activity has a weight of zero, it will not contribute towards the progress of the task at all.
  5. Repeat steps 3-5 to add any further task activities to the task.
  6. To add task activities from a task activity template, select the template in which the task activities are defined in the field beneath the Add buttongrid then select Add From Template.

  7. Click Close.
