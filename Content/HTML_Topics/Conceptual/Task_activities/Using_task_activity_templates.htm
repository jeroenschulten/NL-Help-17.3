Using templates to speed up task activity creation

As well as being able to create task activities individually, you can use task activity templates to speed up the creation of commonly-used sets of task activities.

Creating task activity templates

There are two ways of creating task activity templates: you can either create them from scratch, or create them from a list of task activities that already exists on a task.

To create a task activity template from scratch:

  1. In Library Explorer, display the Task Activity Template library.
  2. Right-click a blank area of the list view, then select New Task Activity Template. A new template is created, with the name initially set to 'Untitled'.
  3. Right-click the template and select Properties. The Task Activity Template Properties dialog appears.
  4. Click Close.

To create a task activity template from an existing list of task activities:

  1. Right-click the task that contains the task activities that you want to save as a template, and select Properties. The Bar and Task Properties dialog appears.
  2. Right-click the Activities grid and select Create Activity Template. The Task Activity Template Properties dialog appears. The grid on the dialog displays the task activities that were associated with the task.
  3. Amend the list of task activities if necessary, then click Close.
 Assigning task activities from task activity template to tasks

To assign the task activities from a task activity template to a task:

  1. Right-click the task and select Properties. The Bar and Task Properties dialog appears.
  2. Select the template in which the task activities are defined in the field beneath the grid then select Add From Template.

  3. Click Close.
