Eigenschappen van de project view bewerken

U kunt de eigenschappen van taken, resources, codes, kostengroepen en kalenders en mijlpaalweergaven weergeven en bewerken door met de rechtermuisknop op het object in de project view te klikken, en vervolgens Eigenschappen te selecteren. Het betreffende dialoogvenster Eigenschappen voor het geselecteerde object wordt weergegeven.

Om hulp te krijgen bij een venster Eigenschappen, drukt u op F1 terwijl u het venster in beeld heeft.
