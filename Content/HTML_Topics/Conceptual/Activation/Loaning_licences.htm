Licentie tijdelijk uitlenen aan een computer

Afhankelijk van het type licentie dat u heeft, kunt u in staat zijn om een licentie van van de ene computer aan een andere computer uit te lenen. Dit is handig als u een licentie van tijdelijk wilt gebruiken op een andere computer. U zou bijvoorbeeld in het weekend op uw thuiscomputer of op een laptop aan een project willen kunnen doorwerken als u niet op kantoor bent. U kunt ook licenties aan een computer uitlenen vanuit een licentie server op uw netwerk.

U kunt de tijdsduur - tot maximaal 90 dagen - opgeven waarin een licentie wordt uitgeleend van de ene computer aan een andere computer. Wanneer een licentie uitgeleend is, kunt u niet gebruiken op de computer waarvan de licentie is uitgeleend. Wanneer de uitleentermijn van de licentie verstreken is kunt u op de donor computer weer gebruiken, maar kunt u niet meer gebruiken op de computer waaraan de licentie was uitgeleend.

Een licentie van een computer aan een andere computer uitlenen:

  1. Installeer en start op de computer waar u een licentie wilt lenen.
  2. Klik op het Bestand tabblad, klik dan op Activeren, selecteer in de Producten groep, en klik dan op Leen licentie. De Uitlenen Licentie wizard verschijnt.

  3. Volg de stappen in de wizard om een uitlenen licentie verzoek code te genereren.
  4. Kopieer de uitlenen licentie verzoek code naar het Windows® Klembord, en breng dit vervolgens over naar de computer waarvan u de licentie wilt lenen, bijvoorbeeld via email.
  5. Op de computer waarvan u de licentie wilt lenen, klik op het Bestand tabblad, klik dan op Activeren, selecteer in de Producten groep, en klik dan op Leen licentie. De Uitlenen licentie wizard verschijnt.
  6. Volg de stappen in de wizard. Wanneer gevraagd, plak de uitlenen licentie verzoek code van de andere computer in de wizard en genereer een licentie code. U moet ook de datum specificeren waarop de licentie-uitleen verloopt.
  7. Kopieer de licentie code naar het Windows® Klembord, en breng dit vervolgens over naar de computer waarnaar u de licentie wilt uitlenen, bijvoorbeeld via email.
  8. Op de computer waaraan u de licentie wilt uitlenen, plakt u de licentie code van de andere computer in de wizard.
  9. Voltooi de wizard om de licentie van de andere computer naar deze computer uit te lenen.
