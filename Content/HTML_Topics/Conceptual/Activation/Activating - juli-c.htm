Activating

Once you have installed , you can use it for a trial period without activating it. You must activate by the end of this period in order to continue using it once the trial period is over. You can activate at any time during this period. Until you activate , an activation reminder appears each time you start the software.

If you have an Internet connection, you can activate automatically over the Internet. If you do not have an Internet connection, you can activate manually with the aid of Elecosoft's Technical Support department. Activating over the Internet is quicker and easier than activating it via the Technical Support department.

To activate :

  1. If you have an Internet connection available, ensure that the connection to the Internet is currently active.
  2. Click the File tab, click Activation, select in the Products group, then click Activate. The Product Activation dialog appears.
  3. Enter details into the fields on the dialog and click OK.

    If your serial number is associated with a standalone licence, a dialog appears to inform you that the activation was successful when the activation is complete. You can now continue to use .
  4. If your serial number is associated with a network licence - one that is contained within a licence server - the Locate Licence Server dialog appears. Enter the name of the computer on which the licence server is located and click OK. Contact your System Administrator for advice if you do not know the name of the computer.

    attempts to contact the licence server on the specified computer and, if a successful connection is made, a dialog appears to inform you that is now configured to obtain its licence from the specified licence server. You can now continue to use .

In some cases, automatic activation may not be successful. Automatic activation will not work in the following circumstances:

  • If you do not have a connection to the Internet, or if your connection to the Internet is not currently active.
  • If Elecosoft's automatic activation service is currently unavailable.
  • If you have installed and are attempting to activate a single copy of on more than one computer.
  • If you are reinstalling on a computer whose hardware has been altered significantly.

If automatic activation fails, a dialog that informs you of the problem will appear and you will either be asked to activate manually - using the Product Activation Wizard - or to contact Elecosoft's Technical Support department for assistance.

Licensing manually using the Product Activation Wizard
  1. Follow the steps in the wizard to generate an activation request code and email it to Elecosoft's Technical Support department.
  2. Once you have transferred the activation request code to Elecosoft, the Technical Support department will send you an activation code. There may be some delay before you receive your activation code. If you wish, you can close the wizard and return to it when you receive the activation code.
  3. When you receive the activation code, reactivate the wizard if necessary and paste the activation code into it, then follow the remaining steps in the wizard to activate .
