Specificeren of een view beschikbaar is in de Project Viewer

U kunt specificeren of de standaard balkkaart viewelke afzonderlijke view in een project beschikbaar is in Elecosofts gratis project viewer software - Project Viewer. Standaard zijn alle viewsis deze view beschikbaar als een project in de Project Viewer wordt geopend. Het zou echter kunnen dat u bepaalde views deze niet beschikbaar maakt in de Project Viewer, bijvoorbeeld wanneer deze view views die gevoelige informatie bevatten, zoals kosten en inkomsten informatie.

Om te specificeren of een viewde standaard balkkaart view beschikbaar is in de Project Viewer:

  1. Geef de standaard balkkaart view weer.
  2. Klik, in het tabblad View, in de Views groep op Eigenschappen. Het View Eigenschappen dialoogvenster - of het Kalender View Eigenschappen dialoogvenster, wanneer de huidige view een kalender view is - verschijnt.
  3. Schakel het selectievakje Beschikbaar in de project viewer in om een view beschikbaar te maken in de Project Viewer, of schakel het selectievakje uit om de view te verbergen voor de gebruikers wanneer het project wordt geopend in de Project Viewer.
