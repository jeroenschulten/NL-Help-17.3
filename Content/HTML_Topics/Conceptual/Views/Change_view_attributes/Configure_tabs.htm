Configureren hoe views worden getoond in tabbladen

U kunt de manier waarop views in tabbladen worden getoond in configureren. U kunt de locatie van de tabbladen configureren en opgeven of u kleuren op de tabbladen wilt toepassen, waardoor u gemakkelijker kunt identificeren welke views bij welk project horen als u meer dan één project tegelijk geopend heeft. U kunt ook configureren of uw gebruikersnaam weergegeven wordt op de view tabbladtitels, in projecten waarin meer dan één gebruiker is gedefinieerd.

Om de manier waarop views worden getoond in tabbladen te configureren:

  1. Klik op het tabblad Bestand, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het Algemeen tabblad, klik vervolgens op de Tabblad Opties knop. Het Tabblad Weergave Opties dialoogvenster verschijnt.
  3. Specificeer of de tabbladen moeten verschijnen boven of onder de views in het Locatie veld.

    Tabbladen tonen onder de views:
  4. Specificeer de wijze waarop de views worden weergegeven in tabbladen - en of de views al dan niet in tabbladen worden weergegeven - in het Tabblad modus veld. Wanneer u er niet voor kiest om de views in tabbladen te tonen, wordt elke view in een apart venster weergegeven.
  5. Schakel het selectievakje Gebruik tabkleuring in om tabkleuring toe te passen op de tabs die verschijnen bovenaan elke view. Zie het gedeelte hieronder voor meer informatie over tabkleuring.

Applying colouring to tabs

If you select the Use tab colouring check box to apply colouring to the tabs that appear at the top of each view, kunt u voor elk project een andere standaardtabbladkleur toepassen, waardoor het gemakkelijker wordt om te identificeren welke views bij welk project horen wanneer u meer dan één project tegelijkertijd hebt geopend. All projects are assigned a default tab colour when you create them, and all views inherit this tab colour from the project.

You can change the default tab colour that is assigned to each project. If you are planning to have more than one project open at a time in , it is useful to apply a different tab colour to each project so that you can easily identify which views belong to which project.

To do this:

  1. Open a project in .
  2. Click the File tab, then click Properties.
  3. Select a colour to apply to the project's tabs in the Default tab colour field.
  4. Save the project.

If you want to apply a different tab colour to one or more specific views, overriding the project's default tab colour, you can do this by applying a colour to a view category and assigning the view category to the views.

To do this:

  1. Either create a new view category, or edit an existing view category.
  2. In the View Category Properties dialog, specify a colour to apply to the tabs in the Tab colour field then click Close.
  3. Assign the view category to one or more views. The tab colour you specified is applied to these views' tabs, overriding the default project tab colour.

To prevent the view category's tab colour from overriding the default project tab colour, edit the view category and select 'None' in the Tab colour field on the View Category Properties dialog.

Om te specificeren of uw gebruikersnaam weergegeven wordt op de view tabbladtitels, bij multi-user projecten:

  1. Klik op het Bestand tabblad, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het View tabblad.
  3. Schakel het Verberg gebruikersnaam in view tabbladtitels
