Object Bewerken werkbalk

U kunt de Object Bewerken werkbalk gebruiken om krachtige bewerkingen uit te voeren op balken, taken, links en toewijzingen. U kunt selecties van balken, taken, links of toewijzingen en combinaties van deze balkkaartobjecten gelijktijdig bewerken.

De Object Bewerken werkbalk omvat de volgende functies:

  • % voortgang
  • Verplaats naar links
  • Verplaats naar rechts
  • Verplaats startdatum naar links
  • Startdatum
  • Verplaats startdatum naar rechts
  • Duur/Link tijdsverschil
  • Verplaats einddatum naar links
  • Einddatum
  • Verplaats einddatum naar rechts
  • Code/Link categorie
  • Kalender

Wanneer u gebruik maakt van , kan het handig zijn om de IFC product en IFC taak type opdrachten toe te voegen aan de Object Bewerken werkbalk. U kunt dit doen door het aanpassen van de werkbalk.
