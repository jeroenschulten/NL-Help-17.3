Understanding the window

has been designed to be easy to use. Its design should be familiar to anyone who has worked with Windows® products before.

The window comprises the following main components:

  • Title bar.
  • Control menu.
  • Ribbon.
  • Backstage view.
  • Quick Access toolbar.
  • Project view.
  • Spreadsheet.
  • Bar chart.
  • Properties view.
  • Object Edit toolbar.
  • Status bar.
  • Library Explorer.
  • Global Library Explorer.
  • Assignment view.
  • Resource usage view pane.
  • Histogram pane.
  • Histogram toolbar.
  • Timeline pane.
  • Line of Balance graph pane.
  • Print preview.
  • Print Preview toolbar.

Note that some components of the window are available only when is used in a particular mode, or at a certain time. For example, the assignment view is visible only when you choose to view it.
