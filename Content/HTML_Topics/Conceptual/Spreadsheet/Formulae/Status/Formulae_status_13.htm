Een context expressie vereist een object-type voorvoegsel

Uitleg bij deze fout:

Deze formule bevat een expressie tussen vierkante haken waaraan een object-type voorvoegsel ontbreekt dat nodig is om te verhelderen naar welk type object (Taak, balk, etc) u wilt verwijzen.

Merk op dat eenvoudige verwijzingen naar baselines zonder een object type voorvoegsel wel toegestaan zijn, zodat een expressie als [~HB] wel geldig is.

[1088]

In dit voorbeeld ontbreekt aan de expressie tussen vierkante haken een voorvoegsel dat het type object bepaalt.

Hoe deze fout op te lossen:

Onderzoek de expressies tussen vierkante haken en zorg dat deze voorafgegaan worden door geldige object-type voorvoegsels.

Wanneer de expressie in het bovenstaande voorbeeld naar een taak zou moeten verwijzen, kan de expressie worden gecorrigeerd met het voorvoegsel Taak, resulterend in Taak[1088].
