Werken met Werkgroepen

Wanneer een nieuw project wordt gemaakt, of een bestaand project wordt geopend, kunt u ervoor kiezen om vanuit een werkgroep met het project bezig te gaan in plaats van als individuele gebruiker.

Wanneer het programma in een werkgroep omgeving wordt gebruikt, delen de gebruikers in dezelfde werkgroep ook de vergrendeling op de objecten. Wanneer een gebruiker begint te wijzigen dan worden de gewijzigde objecten vergrendeld door de gebruiker. Wanneer een gebruiker klaar is met bewerken, blijft het object vergrendeld door de werkgroep zodat andere gebruikers in dezelfde werkgroep dezelfde objecten kunnen bewerken; het is alsof gebruikers van diezelfde werkgroep dezelfde computer gebruiken.

Omdat de gebruikers van de werkgroep samenwerken, geldt ook dat als een gebruiker van de werkgroep het project opslaat, dat alle wijzigingen van de hele werkgroep worden opgeslagen. Overeenkomstig, elke gebruiker van de werkgroep kan de acties van andere gebruikers van de werkgroep ongedaan maken. De lijst 'ongedaan maken' laat de naam zien van de gebruiker die de actie heeft uitgevoerd.

Om de werkgroep te verlaten en het project te wijzigen als een individuele gebruiker, moet u het project verlaten en opnieuw openen zonder een werkgroepnaam op te geven.
