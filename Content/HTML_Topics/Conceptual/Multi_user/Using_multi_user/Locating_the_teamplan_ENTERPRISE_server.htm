De Enterprise server lokaliseren

In een Enterprise omgeving, moet iedere client weten waar de Enterprise server is gelokaliseerd zodat de client verbinding kan maken met de server. U specificeert de methode waarop de server wordt gelokaliseerd door gebruik te maken van het dialoogvenster Client/Server communicatie details.

Om toegang te krijgen tot dit dialoogvenster, klik in het tabblad Bestand op Openen, klik dan op Enterprise en tenslotte op Server Locator.

Wanneer de server locatie instellingen niet zijn gespecificeerd neem contact op met uw systeembeheerder, die u zal vertellen hoe dit te doen.

Deze functionaliteit is niet beschikbaar in de standalone versie van en ook niet in SaaS.

Neem contact op met uw systeembeheerder als zich problemen voordoen met de verbinding met uw server.
