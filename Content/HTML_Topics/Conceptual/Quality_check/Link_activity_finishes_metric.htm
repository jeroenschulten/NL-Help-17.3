'Linkactiviteit eindigt' kwaliteitscriterium

U gebruikt dit kwaliteitscriterium om het aandeel taken te checken dat minstens één uitgaande link heeft die start aan het einde van de taak. Een hoog aandeel nauwelijks-gelinkte taken kan erop wijzen dat de linklogica van de planning foutief is.

Als een balk meer dan één taak bevat, wordt een taak geacht te zijn gelinkt aan zijn onmiddellijke opvolgertaken op de balk door middel van een "impliciete link", zelfs als er geen daadwerkelijke link bestaat tussen de taken, tenzij de balk is geconfigureerd om taken toe te staan te overlappen. Dit kwaliteitscriterium behandelt impliciete links op dezelfde manier als normale links.

De eindtaak van een project zal geen opvolgertaken hebben. U kunt ervoor kiezen om de eindtaak van een project uit te sluiten bij dit kwaliteitscriterium. Schakel het Toestaan eindtaak selectievakje in om de eindtaak uit te sluiten bij dit kwaliteitscriterium.

Als het selectievakje Toestaan eindtaak is ingeschakeld, aangezien er slechts één taak zonder uitgaande links is die aan het einde van de taak begint, zou taak 5 - de eindtaak - worden uitgesloten van dit kwaliteitscriterium; als dit selectievakje zou zijn uitgeschakeld, zou taak 5 worden opgenomen en zou dit kwaliteitscriterium niet slagen.

Slaag en faal criteria

Slaagt of faalt? Criteria
Slaagt

Een project slaagt op dit criterium wanneer het percentage taken met minstens één uitgaande link die start aan het eind van de taak groter is dan of gelijk is aan het slaaggrens percentage.

 

Wanneer bijvoorbeeld de slaaggrens ingesteld stond op 90,00%, zou een project op dit criterium slagen wanneer 90,00% of meer taken minstens één uitgaande link heeft die start aan het eind van de taak.
Faalt

Een project faalt op dit criterium wanneer het percentage taken met minstens één uitgaande link die start aan het eind van de taak kleiner is dan het faalgrens percentage.

 

Wanneer bijvoorbeeld de slaaggrens ingesteld stond op 90,00%, zou een project op dit criterium falen wanneer minder dan 90,00% van de taken minstens één uitgaande link heeft die start aan het eind van de taak.
Slaagt noch faalt Als het percentage taken met minstens één uitgaande link die start aan het eind van de taak valt tussen het slaaggrens en het faalgrens percentage, is het resultaat noch geslaagd noch gefaald, maar ligt dat ergens daartussenin.

Voorgestelde instellingen

  • Toestaan eindtaak: ingeschakeld wanneer het domein het gehele project is; uitgeschakeld wanneer het domein slechts een gedeelte van een project is.
  • Slaaggrens: 90,00%. Een project zal slagen voor dit criterium als 90,00% of meer van de taken minstens één uitgaande link heeft die start aan het eind van de taak.
  • Faalgrens: 90,00%. Een project zal falen voor dit criterium als minder dan 90.00% één uitgaande link die start aan het eind van de taak.
