Wijzigen van (WBN) taak ID's

Taak ID (WBN) codes kunnen worden aangepast vanaf dat ze gegenereerd zijn. De taak ID (WBN) codes kunnen worden ingevoerd en gewijzigd in de tabel of via het tabblad Taak in het dialoogvenster Balk en Taak Eigenschappen. Er kan ook een WBN besturingselement worden toegevoegd aan de Object Bewerken werkbalk, zodat taak ID (WBN) codes kunnen worden ingevoerd en gewijzigd door ze in dit veld in te typen.

Om een taak ID (WBN) code in de tabel te wijzigen:

  1. Klik de cel aan waarvan u de taak ID (WBN) code wilt aanpassen.
  2. Wijzig de code zoals nodig was.
  3. Druk op ENTER of klik of de balkkaart om het wijzigen te beÃ«indigen en de cel niet meer te selecteren.

Om een WBN code te wijzigen op het tabblad Taak van het dialoogvenster Balk en Taak Eigenschappen :

  1. Vraag het dialoogvenster Balk en Taak Eigenschappen op voor de taak die u wilt aanpassen.
  2. Klik op het tabblad Taak.
  3. Klik op Details in het linkerpaneel van het tabblad.
  4. Pas indien gewenst het veld Taak ID (WBN) code aan.

Wanneer u de Work Breakdown Structure hernummert, worden taak ID (WBN) codes die u hebt bewerkt overschreven door de nieuwe taak ID (WBN) codes.
