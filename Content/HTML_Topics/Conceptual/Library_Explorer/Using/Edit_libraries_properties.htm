Bibliotheekobjecten bewerken met behulp van het dialoogvenster Eigenschappen

De meeste bibliotheekobjecten beschikken over een dialoogvenster Eigenschappen, dat u kunt gebruiken om hun eigenschappen te bewerken. Sommige complexe bibliotheekobjecten, zoals kalenders en filters, kunnen alleen volledig worden bewerkt met het behulp van het betreffende dialoogvenster Eigenschappen.

Om het dialoogvenster Eigenschappen weer te geven voor een object kiest u één van de volgende stappen:

  • Klik met de rechtermuisknop op een object of een map en kies vervolgens de opdracht Eigenschappen.
  • Selecteer een object of map en druk vervolgens SHIFT F2 of ALT ENTER.
  • Dubbelklik op een object in de lijst view (afhankelijk van uw standaard opties, zal bij dubbelklik op een map, de map worden geopend of gesloten worden).

De opties in het dialoogvenster Eigenschappen variëren afhankelijk van het type object dat u aan het bewerken bent. Toets F1 in het dialoogvenster Eigenschappen om de gedetailleerde help van het dialoogvenster weer te geven.
