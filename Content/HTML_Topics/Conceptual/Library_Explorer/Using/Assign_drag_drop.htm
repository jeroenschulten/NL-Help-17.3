Bibliotheekobjecten toewijzen met behulp van slepen en neerzetten

U kunt bibliotheekobjecten (en mappen) toewijzen aan taken door ze uit Bibliotheek verkenner te slepen en ze te plaatsen op taken in de balkkaart. U kunt bijvoorbeeld resources en kalenders op deze manier toewijzen.

Wanneer u op met slepen en neerzetten een resource of een kostengroep toewijst, is de toewijzing gebaseerd op de standaard instellingen van de resource of de kostengroep. U kunt de resourcetoewijzing of de kostengroepallocatie nog bewerken nadat deze zijn toegewezen.

Wanneer u een resource, kostengroep, toewijzingsgroep of code met slepen op een lege balk regel neerzet, dan wordt er een nieuwe taak aangemaakt. Het resource, de kostengroep, toewijzingsgroep of code wordt toegewezen aan de taak, waarbij de taak de standaard duur krijgt. Als het een gemodelleerde resource is wordt de taak duur bepaald door de toewijzing.

U kunt meer dan één code van dezelfde codebibliotheek aan een object toewijzen. Om van dezelfde codebibliotheek meer als één code toe te wijzen aan een object, sleept u de code uit de Bibliotheek verkenner en drukt u CTRL in terwijl u het neerzet op een object.

Sommige bibliotheekobjecten (zoals codes, kalenders, resources en kosten) kunt u ook toewijzen door met de rechtermuisknop op een taak te klikken en in het pop-up menu de betreffende opdracht Toewijzen te kiezen. Wanneer er meer dan 25 items zijn waaruit u kunt kiezen, kiest u de opdracht Meer om het object te selecteren uit een dialoogvenster.
