Wat is de Bibliotheek verkenner?

Een project kan vele verschillende typen objecten bevatten zoals hulpmiddelen, kalenders, codes, kostengroepen, munteenheden en gebruikers. Deze en andere objecten worden bewaard in bibliotheken. De Bibliotheek verkenner toont alle bibliotheekobjecten in een project. U gebruikt de Bibliotheek verkenner voor het beheren van bibliotheekobjecten. Dit gedeelte legt uit hoe u de Bibliotheek verkenner en de verschillende bibliotheekobjecten kunt gebruiken.

In een groepsproject, worden bibliotheekobjecten gedeeld met alle andere projecten in dezelfde groep, dus hulpmiddelen die gemaakt zijn in één project kunnen gebruikt worden in een ander project binnen dezelfde groep.
